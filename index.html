The Tasklist

  1. Be able to add primitive drawing objects to the drawing:
    • a circle
    • rectangle
    • line
    • text
    • freehand drawing by a pen
  2. The application should support undo (and preferably redo as well)
  3. Bonus points: the application should support math formulas
  4. Should be possible to save and load a drawing
  5. Add color selector - http://jweir.github.com/colorwheel/
  6. All elements should be movable. Preferably it should also be possible to move many objects simultaneously
  7. Should be possible to group these primitives together into a template, which can be saved and then added to a new whiteboard with ease
